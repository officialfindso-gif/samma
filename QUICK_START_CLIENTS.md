# 👥 Управление клиентами - Быстрый старт

## Что нового?

Теперь вы можете управлять несколькими клиентами, отслеживать историю взаимодействий и быстро переключаться между проектами!

## 🚀 Как начать?

### 1. Создайте первого клиента

1. Перейдите в раздел **"👥 Клиенты"** (ссылка в меню или `/app/clients`)
2. Нажмите **"➕ Добавить клиента"**
3. Заполните форму:
   - **Название проекта** *(обязательно)* - например: "Сайт для кофейни"
   - **Название клиента** - ИП Иванов или ООО "Ромашка"
   - **Контакт** - email, телефон или Telegram
   - **Заметки** - любая важная информация
   - **Цвет** - выберите цвет для визуального различения
   - **Теги** - например: `vip, срочно, активный` (через запятую)
4. Нажмите **"➕ Создать"**

### 2. Работайте с постами клиента

**Способ 1: Быстрое переключение**
- Нажмите на иконку клиента в шапке сайта (справа вверху)
- Выберите нужного клиента из списка
- Вы автоматически перейдёте к его постам

**Способ 2: Через карточку клиента**
- Откройте раздел "👥 Клиенты"
- Нажмите **"🎯 Посты"** на нужной карточке

### 3. Отслеживайте активности

**Где:** Откройте детальную страницу клиента (кнопка **"📊 Подробнее"**)

**Типы активностей:**
- 📝 **Заметка** - важная информация
- 📞 **Звонок** - телефонный разговор
- 🤝 **Встреча** - личная встреча или видеозвонок
- 📧 **Email** - переписка по почте
- ✨ **Создан пост** - когда создали новый контент
- ✅ **Утверждён пост** - клиент одобрил пост
- 💰 **Оплата** - получена оплата
- 📌 **Другое** - любые другие взаимодействия

**Как добавить:**
1. Нажмите **"➕ Добавить"** в блоке активностей
2. Выберите тип активности
3. Введите заголовок (например: "Обсудили новую кампанию")
4. Опционально добавьте описание
5. Нажмите **"➕ Создать"**

## 💡 Полезные советы

### Используйте теги для организации

**Примеры тегов:**
- `vip, срочно` - для приоритетных клиентов
- `активный, 10постов/месяц` - по объёму работы
- `архив, завершён` - для завершённых проектов
- `новый, тестовый` - для новых клиентов

### Выбирайте цвета для быстрой навигации

- **Красный** - срочные проекты
- **Зелёный** - стабильные клиенты
- **Синий** - новые проекты
- **Фиолетовый** - VIP клиенты

### Регулярно добавляйте активности

Записывайте каждое важное взаимодействие:
- ✅ После звонка с клиентом
- ✅ После отправки материалов
- ✅ При получении оплаты
- ✅ После встречи или презентации

## 📊 Что показывает dashboard клиента?

На странице детального просмотра (**"📊 Подробнее"**) вы увидите:

1. **Статистика:**
   - 📝 Сколько постов создано
   - 👥 Количество мест в команде
   - 📅 Сколько дней работаете с клиентом
   - ⚡ Количество активностей

2. **Заметки:**
   - Полный текст ваших заметок о клиенте

3. **История активностей:**
   - Все взаимодействия с клиентом
   - С датами, типами и описаниями
   - Можно удалить старые записи (наведите на активность)

4. **Информация о проекте:**
   - Когда создан workspace
   - Когда последний раз обновлялся
   - Цвет и ID для технической поддержки

## 🎯 Типичный рабочий процесс

### Новый клиент

1. **Создайте клиента** с полной информацией
2. **Добавьте теги** для категоризации
3. **Создайте первый пост** через быстрый переключатель
4. **Записывайте активности** при каждом взаимодействии

### Еженедельная работа

1. **Используйте быстрый переключатель** для навигации между клиентами
2. **Создавайте посты** для текущего клиента
3. **Добавляйте активности** после звонков и встреч
4. **Проверяйте dashboard** для анализа работы

### Ежемесячный отчёт

1. **Откройте страницу клиента**
2. **Посмотрите статистику** - сколько постов создано
3. **Просмотрите активности** - что было сделано за месяц
4. **Обновите заметки** с важной информацией

## ⚙️ Управление клиентами

### Редактирование

- Откройте раздел "👥 Клиенты"
- Нажмите **"✏️ Редактировать"** на карточке
- Измените нужные поля
- Сохраните изменения

### Удаление

- Нажмите **"🗑️"** на карточке клиента
- Подтвердите удаление
- ⚠️ **Внимание:** Посты останутся, но workspace будет удалён

## ❓ FAQ

**Q: Как найти нужного клиента быстро?**
A: Используйте быстрый переключатель в шапке - там все клиенты с цветными метками.

**Q: Можно ли фильтровать клиентов по тегам?**
A: Пока нет, но эта функция в планах. Сейчас теги видны на карточках для визуальной навигации.

**Q: Что будет с постами при удалении клиента?**
A: Посты останутся в системе, но workspace (проект) будет удалён. Будьте осторожны!

**Q: Можно ли изменить тип активности после создания?**
A: Пока нет редактирования активностей, только удаление и создание новой.

**Q: Сколько клиентов можно создать?**
A: Неограниченное количество. Система протестирована с 10+ клиентами.
