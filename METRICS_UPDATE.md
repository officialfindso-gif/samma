## Что добавлено

### Backend (✓ Готово)
1. **Новые поля в модели Post:**
   - `views_count` - количество просмотров
   - `likes_count` - количество лайков
   - `comments_count` - количество комментариев
   - `shares_count` - количество репостов/шеров
   - `engagement_rate` - процент вовлеченности (%)

   - `published_at` - дата публикации оригинального контента

2. **Scraper обновлен:**
   - Извлекает метрики из Instagram API
   - Автоматически вычисляет engagement rate
   - Конвертирует Unix timestamp в datetime

3. **API сериализатор:**
   - Все новые поля доступны через REST API
   - Marked as read-only (только для чтения)

### Frontend (✓ Готово)

## Как протестировать

1. **Обновите браузер** на странице http://localhost:3000
   - Должны появиться 4 новые колонки в таблице

2. **Запустите Celery worker** (если еще не запущен):
   ```bash
   cd backend
   start_celery.bat
   ```

3. **Создайте новый пост** или **Переобработайте существующий**:
   - Пост #10 уже сброшен в статус 'new'
   - Нажмите "Обработать" в UI

4. **Дождитесь обработки** (30-60 секунд):
   - Парсинг Instagram → получение метрик
   - LLM генерация → создание контента

5. **Проверьте результаты:**
   - В таблице появятся цветные цифры метрик
   - В детальном виде - полный набор карточек с метриками
